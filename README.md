# Portfolio

Site statique personnel de Bill Adjagboni.

Structure du projet:

- `index.html` — page principale
- `assets/css/` — feuilles de style
- `assets/js/` — scripts
- `assets/img/` — images (portrait)
- `assets/docs/` — documents (CV)

Pour prévisualiser localement, ouvrez `index.html` dans un navigateur ou servez avec un serveur statique.
